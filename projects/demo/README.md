# HSS Calendar Demo App

This is a full-featured demonstration application for the `hss-calendar` library. It showcases the library's versatility, modern design, and ease of integration.

## ðŸŒŸ Key Features of the Demo
- **Event Lifecycle**: Add, Edit, and Delete events using a premium modal UI.
- **Date Range Support**: Create events with separate start and end dates.
- **Real-time Theming**: Toggle between Light and Dark modes instantly.
- **Dynamic Sidebar**: View a detailed list of all events for any clicked date.
- **Color Selection**: Assign custom colors to events using a color picker.

## ðŸš€ How to Run

1. **Navigate to the root directory**.
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Build the library** (or run in watch mode):
   ```bash
   npm run lib:watch
   ```
4. **Start the demo**:
   ```bash
   npm start
   ```
5. **Open locally**: Navigate to `http://localhost:4200`.
